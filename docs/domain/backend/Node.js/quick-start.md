## Node.js Module（模块）

Node.js 初学者比较容易困惑的主题之一就是 Node.js 的模块系统，深入理解它对你的 JavaScript 和 Node.js 编程至关重要。

Node.js 有两种模块系统：
- 早期的 [CommonJS 模块](http://nodejs.cn/api/modules.html)
- 现在主流的 [ECMAScript 模块](http://nodejs.cn/api/esm.html)
